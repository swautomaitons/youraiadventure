Analysis

The performance gap that justified closed-model API pricing has closed. This is a strategic inflection point: the open vs closed debate is now a cost-benefit analysis, not a capability discussion. Teams running more than 2-5M tokens/month have a clear economic case for self-hosting. Below that threshold, managed APIs still win on simplicity. The honest remaining advantages of closed APIs — managed infrastructure, built-in safety, compliance documentation — are real but no longer include performance.
