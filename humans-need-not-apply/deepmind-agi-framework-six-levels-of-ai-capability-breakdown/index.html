Analysis

DeepMind published a cognitive framework for evaluating AGI milestones this week.

The core argument: the field has been measuring proxies, not AGI itself. Benchmarks like MMLU and ARC test specific capabilities. DeepMind's framework tests generalization across cognitive domains — and those are different things.

Five signals worth tracking:

1. Performance ≠ generalization — the framework explicitly decouples these for the first time at this scope

2. Crowdsourced validation — DeepMind launched a Kaggle hackathon for community-built benchmarks. A defining framework that needs external validators signals epistemic humility about what AGI measurement actually requires

3. AlphaGo's 10-year arc — Go → protein folding → drug discovery. The pattern: each domain breakthrough was a preview of the next

4. Gemini 3.1 Flash-Lite — speed-at-scale deployment alongside the framework isn't coincidence. Research + infrastructure are moving in parallel

5. The measurement problem is now officially on the table — and that changes how the industry builds toward AGI

The question worth asking: if current benchmarks are measuring proxies, what decisions are being made based on those benchmarks?
